# 💳 Credit Card Analysis Dashboard

## 📌 Overview

The Credit Card Analysis Dashboard is a Business Intelligence project developed using **Python, SQL, Power BI, and DAX**. This project analyzes customer and transaction data to identify spending patterns, customer behavior, revenue trends, and key business insights through interactive dashboards.

---

## 🚀 Project Workflow

- Data Cleaning and Preprocessing using Python (Pandas, NumPy)
- Data Analysis using SQL
- Data Modeling and DAX Measures
- Dashboard Development using Power BI
- Interactive Data Visualization and Reporting

---

## 🛠️ Technologies Used

- Python (Pandas, NumPy)
- SQL
- Power BI
- DAX
- Power Query

---

## 📊 Dashboard Features

- Revenue Analysis
- Customer Segmentation
- Credit Card Transaction Analysis
- Customer Demographics
- Card Category Performance
- Weekly & Quarterly Revenue Trends
- Interest Earned Analysis
- Interactive KPI Cards
- Dynamic Filters & Slicers

---

## 📈 Key Insights

- Identified customer spending patterns and revenue trends.
- Analyzed transaction performance across different customer segments.
- Compared card category performance using interactive visualizations.
- Monitored key business KPIs for better decision-making.
